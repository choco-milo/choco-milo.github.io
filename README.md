## Running Locally

### Prerequisites
- Node.js 18+
- npm or yarn

### Steps

1. Clone the repository:
```bash
git clone https://github.com/choco-milo/choco-milo.github.io
cd choco-milo.github.io
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Updating Content

All personal information lives in one place ‚Äî `data/profile.json`. Edit that file to update your bio, skills, experience, projects, education, and certifications. The site updates automatically.

### Building for Production

```bash
npm run build
```
